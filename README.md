# Школьный веб-портал

Мой первый учебный проект, разработанный за ~10 дней

## Описание проекта
Веб-сайт для образовательного учреждения с комплексной системой управления контентом.

## Основные возможности

1. **Конструктор новостей**  
   - Система публикации новостей с блочным принципом формирования контента (аналогично сообщениям в Discord)
   
2. **Адаптивный интерфейс**  
   - Корректное отображение на различных устройствах

3. **Система аутентификации**  
   - Регистрация и авторизация пользователей

4. **База преподавателей**  
   - Карточки преподавателей с поиском по базе

5. **Форма обратной связи**  
   - Возможность задать вопрос или отправить заявку на поступление

6. **Экспорт данных**  
   - Выгрузка вопросов и заявок в виде архива (для пользователей с соответствующими правами доступа)

7. **Централизованное хранение данных**  
   - Вся информация (преподаватели, новости, контакты, дипломы, лицензии, вопросы и заявки) формируется из базы данных

## Используемые технологии:
- Django, HTML5, CSS3, JavaScript, Bootstrap
